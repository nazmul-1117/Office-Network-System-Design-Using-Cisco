## Requirements:
1. Routing Protocol
2. OSPF (Open Shortest Path First)
3. FTP (File Transfer Protocol)
4. TFTP (Trivial File Transfer Protocol)
5. NAT (Network Address Translation)
6. PAT (Port address translation)(Optional)
